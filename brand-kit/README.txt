MARIELL — Système de marque · v1.1
=======================================

Logo validé : Tiempos Headline Medium 500 · CASSE: MARIELL (uppercase)
Intensité chromatique : 45 %
Couleurs : #5EE7E7 (cyan) · #E85EFF (magenta) · #FFFFFF (white)
Fond canonique : #050507
Animation : burst-glitch · cycle 5,5 s · 2 bursts (16-26 % et 80-91 %)

Contenu :
01-sources/        — masters vectoriels (SVG + PDF)
02-color/          — wordmark couleur, fond noir + fond clair
03-monochrome/     — wordmark blanc et noir
04-favicons/       — SVG + ICO + Apple + Android
05-email-signature/— PNG @2× standard + dark-mode
06-social/         — photos de profil carrées

Notes :
- Les SVG embarquent la fonte "Tiempos Headline" en base64 (autonomes).
  Aucune dépendance externe : le rendu est identique partout.
  Pour figer la marque sur un import Illustrator,
  installer la fonte ou convertir les textes en tracés (Calque > Vectoriser).
- Le PDF est rasterisé en interne (JPEG haute qualité) — pour un
  master strictement vectoriel, ouvrir le master.svg dans Illustrator
  et enregistrer comme .ai / .eps / .pdf.
- L'ICO contient 16/32/48 px PNG-in-ICO (compatible Windows Vista+).
