## Cadre méthodologique

### Définition

Une "boîte intouchable" est une entreprise dont **la marque, le produit, la trajectoire ou le positionnement créent une force d'attraction naturelle sur le marché Sales**. Concrètement : ces entreprises attirent des candidats Sales **sans avoir à compenser par un package supérieur au marché**. La désirabilité vient de ce qu'elles sont, pas de ce qu'elles paient.

### Principe économique sous-jacent

Une boîte intouchable peut afficher un OTE médian (voire en-deçà) et rester très attractive parce que la marque et le produit suffisent à créer le désir. À l'inverse, une boîte non-intouchable doit afficher un OTE supérieur au marché pour compenser son déficit d'attractivité naturelle. **L'attractivité d'une offre est un produit de l'attractivité de la boîte multiplié par le package — pas une somme.**

### Les 4 tiers

| Tier | Force d'attraction | Bonus appliqué | Phraseo type prospect |
|---|---|---|---|
| **S** | Aspirateur absolu — l'attractivité dépasse le secteur | **Bonus énorme** — capital fort automatique, score variable au sein du tier | *"Position hyper préférentielle sur le marché Sales. Attractivité naturelle forte tirée par votre produit et votre marque."* |
| **A** | Aspirateur sectoriel — référence dans son secteur | **Bonus important** — capital fort, **constant** peu importe le secteur du candidat ciblé | *"Position de référence dans votre secteur. Vous êtes identifiés comme une cible désirable par les Sales qui connaissent votre marché."* |
| **B** | Signature crédible — solide, identifiable, sans aspiration | **Bonus modéré** — l'offre concrète pèse plus que la marque | *"Signature crédible et identifiable. La différenciation passera prioritairement par votre offre : package, mission, équipe, évolution."* |
| **C** | Reconnaissance par exposition — visibilité réseaux sans effet d'aspiration | **Bonus très léger** — biais de reconnaissance, légèrement mieux qu'inconnu | *"Présence identifiable sur le marché. Pas de marque dominante mais une visibilité qui rassure légèrement."* |

### Règles fondamentales d'usage

> ⚠️ **PRINCIPE DIRECTEUR TRANSVERSAL — RESSOURCE D'AIDE, JAMAIS DE FREIN**
>
> Ce document (comme les Fichiers 2, 3 et 4) est une **ressource pour améliorer la précision de l'évaluation**, jamais pour la freiner.
>
> - Si une info présente dans ce fichier permet de qualifier plus précisément l'entreprise du prospect → **applique le bonus** (Tier S/A/B/C).
> - Si l'info n'est pas trouvable ou si l'entreprise n'apparaît pas dans le document → **pas de malus**, pas de blocage. L'évaluation se poursuit normalement sur les autres dimensions (package, équipe, missions, croissance, secteur).
> - Le LLM ne perd jamais ses moyens face à un manque d'info. Il rend toujours une évaluation, même partielle.
>
> **Spécificité du Fichier 1 — bonus uniquement, jamais malus** :  
> Une entreprise ABSENTE de ce document n'est JAMAIS pénalisée. Elle est évaluée objectivement sur les autres dimensions. L'absence du document n'est jamais un signal négatif.
>
> Le LLM doit traiter une PME inconnue avec la même rigueur analytique qu'une licorne du Tier S. Le bonus ne se substitue pas à l'analyse, il s'y ajoute.

**Match nominal direct** — Si l'entreprise du prospect est dans la liste, le tier conditionne uniquement le **plancher d'attractivité** de l'éval. Tous les Tier S ne reçoivent pas le même score final — le bonus est un seuil, pas un score.

**Match contextuel** — Si l'entreprise du prospect n'est pas dans la liste mais est **concurrent direct** d'une intouchable, le LLM note que le prospect joue dans une cour exigeante (mais sans malus pour autant).

**Match négatif (absence pure)** — Si l'entreprise n'est ni intouchable ni concurrent direct, l'évaluation se fait **strictement objectivement** sur les autres dimensions. **Aucune pénalité d'absence.**

### Règle absolue de communication

Le LLM ne nomme **jamais** les tiers (S/A/B/C). Il ne mentionne pas la liste. Il traduit le tier en langage métier via les phraseo types (à adapter à chaque cas, pas à réciter).

### 5 critères d'inclusion (au moins 3 sur 5 requis)

1. Funding crédible et récent (Series B+ ou rentabilité affichée publiquement)
2. Équipe Sales structurée (≥10 Sales identifiables)
3. Marque employeur Sales positive (Glassdoor ≥ 3.8 / RepVue ≥ 75)
4. Visibilité durable sur le marché Sales français
5. Présence VP Sales / CRO crédible et identifiable

### Exclusions strictes

- **B2C dominant** (sauf marque exceptionnellement forte → Tier C autorisé : Finary, Ramify, Trade Republic)
- Boîtes en plan social majeur ou liquidation
- Glassdoor < 3.5

## Note méthodologique pour intégration prompt LLM

Quand ce référentiel sera injecté dans le system prompt de l'outil 3, il devra être accompagné des consignes suivantes :

1. **Bonus uniquement** — l'absence d'une entreprise du document n'entraîne aucun malus. Évaluation objective sur les autres dimensions.
2. **Tier non nommé** — le LLM ne mentionne jamais les noms de tiers (S/A/B/C) ni l'existence d'une liste interne au prospect.
3. **Phraseo type** — le LLM utilise les formulations d'attractivité du tier comme inspiration, jamais en récitation.
4. **Match exact requis** — éviter les correspondances approximatives (ex : "Bib Batteries" ≠ "Big Batteries"). En cas de doute sur l'identité de l'entreprise du prospect, le LLM traite comme absente.
5. **Bonus en seuil, pas en score** — le tier conditionne un plancher d'attractivité, mais le score final est modulé par les autres dimensions (package, missions, équipe, croissance).
